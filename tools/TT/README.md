# Typing Tutor IV

打字教学程序，原名 **Typing Tutor IV with Letter Invaders**。

- 开发商：Kriya Systems
- 发行商：Simon & Schuster
- 发布年份：1987
- 平台：MS-DOS

## 文件列表

| 文件 | 大小 | 说明 |
|------|------|------|
| TT.EXE  | 100,912 bytes | 主程序 |
| TT.HLP  | 12,508 bytes  | 帮助文件 |
| TT.HIS  | 18,319 bytes  | 学习历史记录 |

## 功能

- 指法练习课程
- 打字测试
- 进度报告与统计图表
- Letter Invaders 打字游戏
- 数字小键盘练习
- 自定义磁盘测试
- 多人学习记录

## 操作

- `Esc` - 返回 / 退出当前功能
- `Space` / `Enter` - 完成一行练习
- `Backspace` - 纠正错误
- 菜单导航 - 方向键选择，`Enter` 确认

## 运行

在 DOS 环境或 DOSBox 中执行 `TT.EXE` 即可启动。
