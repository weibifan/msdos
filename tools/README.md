# DOS Tools Collection

Useful DOS utilities for retrocomputing.

| File | Description |
|------|-------------|
| `ARJ.EXE` | ARJ v2.50 — file archiver by Robert K. Jung. High-compression archiver with multi-volume support, widely used in the 1990s. |
| `pkzip250.exe` | PKZIP v2.50 — file archiver by Phil Katz (PKWARE). Industry-standard ZIP compression with fast performance and broad compatibility. |
| `pct9.zip` | PC Tools 9.0 — comprehensive DOS utility suite by Central Point Software. Includes file manager (PC Shell), disk editor (DiskEdit), recovery tools (Undelete, Unformat), disk diagnostic (DiskFix), system optimization (SI Pro, Compress), and more. |
| `sea13.zip` | Sea v1.3 — fast VESA-compatible image viewer/converter by Photodex Corporation (1997). Views PNG, GIF, JPEG, TARGA, BMP, PSD, TIFF, PCX, LBM, PNM, CEL, WPG and more. Plays AVI, FLI, FLC animations. Mouse-controlled GUI. |
| `TT/` | Typing Tutor IV — typing tutor program with Letter Invaders game by Kriya Systems (1987). |
