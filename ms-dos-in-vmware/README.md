# MS-DOS 6.22 in VMware Workstation 5.5

Pre-configured VMware virtual machine with MS-DOS 6.22 installed.

## Files

- **ms-dos-vmware-5.5.zip** — Compressed VM package containing:
  - Virtual hard disk with MS-DOS 6.22 installed
  - VMware configuration file (.vmx)
  - BIOS and VM settings optimized for MS-DOS

## Quick Start

1. Extract `ms-dos-vmware-5.5.zip`
2. Open the `.vmx` file with **VMware Workstation 5.x** or later
3. Power on the virtual machine

## VM Specifications

| Component | Detail |
|-----------|--------|
| CPU | 1 vCPU |
| RAM | 64 MB |
| Storage | 200 MB IDE disk |
| Networking | None (MS-DOS has no native TCP/IP) |
| Sound | PC speaker |
| Graphics | Standard VGA |
| Floppy | 1.44 MB drive |

## Notes

- Compatible with VMware Workstation 5.x, 6.x, and later versions
- Can also be imported into VirtualBox (may need configuration tweaks)
